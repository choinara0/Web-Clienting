분식 및 간식

    NAME : 신전떡볶이
    LOCATION : 서울 성북구 솔샘로4길 46 (정릉시장 쓰리팝건물 옆!)
    TIME : 11:00 ~ 22:00 (금요일 휴무)
    REVIEW : 정릉동 신전떡볶이는 다른 곳에 있는 신전떡볶이 보다 양이 많다. 매콤하기 때문에 참치마요컵밥이나 신전김밥과 함께 먹는 것을 추천한다.
                    신전떡볶이 특유의 오뎅튀김은 바삭할 때 가장 맛있기 때문에 배달을 시키는 것보다 직접 가서 먹는 것이 맛있다.
	
    NAME : 전설의 사발떡볶이
    LOCATION : 서울 성북구 정릉로6길 45
    TIME : 11:00 ~ 21:00
    REVIEW : 국민대학교 맛집 중 하나인 전설의 사발떡볶이. 학생들이 즐겨 찾는 이유는 가격 대비 양이 많다.
                    돈까스 사리와 함께 먹으면 정말 맛있다.
	
    NAME : 국민이네 떡볶이
    LOCATION : 서울 성북구 정릉로10나길 2(지하세계 파출소 옆!)
    TIME : soon update
    REVIEW : 다른 떡볶이 집과는 달리 크림누들떡볶이, 순대볶음떡볶이 등 떡볶이의 종류가 다양하다.
                    초등학교 때 분식집을 갔던 기분을 다시 느낄 수 있는 곳.
	
    NAME : 이삭토스트
    LOCATION : 서울 성북구 정릉로9길 36(국민대학교 후문)
    TIME : soon update
    REVIEW : 국민대학교 근처에는 subway가 없어서 토스트를 즐길 수 있는 곳은 이삭토스트 뿐이다.
                    이삭토스트의 장점이라면 간단하게 즐길 수 있고 저렴하다는 것이다. 양은... 간식용이다.
                    자주 가면 사장님께서 알아봐주시고 친절하게 인사도 해주신다.
	
    NAME : Place N
    LOCATION : 국민대학교 복지관 2층
    TIME : soon update
    REVIEW : 국민대학교의 명물 빵집 'Place N' 이다.
                    복지관에 위치하기 때문에 신입생들은 잘 모를 수도 있지만 한 번 가게 되면 한 달에 최소 3번은 간다.
