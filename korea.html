    NAME : 채움늘
    LOCATION : 서울 성북구 정릉로9길 40(국민대학교 후문)
    TIME : soon update
    REVIEW : 담백하고 깔끔한 맛의 볶음밥을 파는 집이다. 6000원대의 가격대며 양이 생각보다 많다.
                    만두와 같이 먹으면 더욱 맛있는데 사장님께서 손님 수에 맞게 더 많이 주신다.
                    밑반찬으로 나오는 사골국은 3번 정도 먹어야 한다.
	
    NAME : 작은마을
    LOCATION : 서울 성북구 정릉로6길 35 (정문에서 지하세계로 내려오면 바로 있다!)
    TIME : soon update
    REVIEW : 국민대학교 근처에는 작은 마을 외에도 제육덮밥을 파는 집이 많다.
                    하지만 작은마을은 다른 가게와는 달리 덮밥 위에 계란후라이를 올려준다.
                    이 차이가 굉장히 크다. 제육덮밥을 좋아하는 사람이라면 알 것이다.
	
    NAME : 보리보리
    LOCATION : 서울 성북구 동소문로 44길 43(소한마리 근처에 있다!)
    TIME : 11:30 ~ 21:30
    REVIEW : 학교 근처에 몇 없는 웰빙식당이다.
                    보리나물 정식, 청국정식 등 맛있고 몸도 건강해지는 듯한 메뉴들이 있다.
                    보리나물 제육정식과 황태정식을 추천한다.
	
    NAME : 현주네 포차
    LOCATION : 서울 성북구 정릉로10길 36
    TIME : 15:00 ~ 01:00 (일요일 휴무)
    REVIEW : 현주네 포차의 시그니처 메뉴는 닭볶음탕이다.
                    大자를 시키면 계란후라이를 주시는데 이 계란은 아껴뒀다가 볶음밥에 비벼먹는 것을 추천한다.
                    워낙 양이 많고 맛있어서 학기 초나 시험 직후에는 과나 동아리에서 단체로 오기 때문에 시끄럽다.
